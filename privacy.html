Privacy Policy

MaruComprehensible Chrome Extension  ·  Last updated: June 2025

Short version: MaruComprehensible stores your MaruMori API token locally on your device and never transmits it to anyone other than MaruMori. No browsing data, no analytics, no ads.

What data we collect

MaruComprehensible stores the following data locally on your device using chrome.storage.local:

  • Your MaruMori API token, which you paste voluntarily to connect your account
  • Your vocab and kanji lists, downloaded from MaruMori and cached locally so scoring works offline
  • UI preferences (theme, font size, sort order)

This data never leaves your device except as described below.

Network requests

The extension makes the following outbound requests:

  • MaruMori API (public-api.marumori.io) — to fetch your vocab and kanji lists using your API token. This request is made only when you click "Connect" or "Refresh" in the popup.
  • YouTube timedtext API (youtube.com) — to fetch Japanese subtitle tracks for the video currently playing, for comprehension scoring. No account credentials are sent; only a public caption URL is fetched.
  • Jisho dictionary API (jisho.org) — to look up word definitions when you hover over a word. Only the hovered word is sent.

No other external servers are contacted. The extension does not use analytics, crash reporting, or advertising services.

Data sharing

We do not sell, share, or transmit your data to any third party. The only external services the extension communicates with are those listed above, solely to provide the extension's core functionality.

Data stored on your device

All data is stored locally via the Chrome storage API and is never synced to any remote server controlled by this extension. You can clear all stored data at any time by clicking "Disconnect" in the extension popup, or by removing the extension from Chrome.

Permissions used

  • storage — to save your API token and preferences locally
  • activeTab — to read the current tab's URL so the popup can show the correct score
  • scripting — to fetch YouTube caption files using the page's session cookies (required because YouTube rejects requests from extension origins)
  • Host permissions — limited to the specific sites the extension supports (CIJ, Nihongo-Jikan, YouTube, MaruMori API, Jisho)
